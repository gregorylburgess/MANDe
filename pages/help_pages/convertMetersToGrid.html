convertMetersToGridR Documentation

Converts input parameters from meters to grid cells.

Description

These are used for internal calculations and are invisible to the user.

Usage

  convertMetersToGrid(params, bGrid)

Arguments

params

A dictionary of parameters, see PARAMETER_DESCRIPTIONS.html for more info.

bGrid

A valid bGrid.

Value

The 'params' parameter, populated with necessary internal variables ("sd", "suppsd", "range", "suppressionRange") with unit 'grid cells'.
