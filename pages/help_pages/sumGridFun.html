sumGridFunR Documentation

Calculates the composite "goodness" grid for a particular bias.

Description

Calls a particular sumGrid function based on the bias and opt values. Actual work is done by the called function.

Usage

  sumGridFun(grids, range, bias, params, debug = FALSE,
    silent = FALSE)

Arguments

bias

The goodness algorithm to use, choose 1, 2, or 3. See above for descriptions.

grids

A dictionary containing the keys 'bGrid', 'fGrid', and 'sumGrid', which hold a valid BGrid, FGrid and SumGrid.

range

The range of the sensor in bathymetric cells.

params

A dictionary of parameters, see PARAMETER_DESCRIPTIONS.html for more info.

debug

If enabled, turns on debug printing (console only).

silent

If set to TRUE, disables status printing.

Value

Returns the grids parameter, with an updated sumGrid.
