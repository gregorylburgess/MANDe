appendErrorR Documentation

Appends execution errors to a global gErrors dictionary, with the 'time' variable as the key.

Description

Creates an entry in the global 'gErrors' dictionary with the 'time' value as a key, and msg as a value, then prints msg to the terminal.

Usage

  appendError(msg, time)

Arguments

msg

The error message to print.

time

A timestamp to use as a key.

Value

The error message that was passed in.
